### ðŸ’» How to set up AI Speed Dial as your landing page
#### Edge
- Go to [Settings](edge://settings)
- On the left sidebar, hit **Start, home, and new tab page**
- From there, navigate to the **On startup** section
- Select the **Open custom sites** buton
- Hit **Add site**
- Enter the URL **https://shankar-sachin.github.io/ai-speed-dial** into the text box
- Refresh Edge to see your beautiful new landing page!
#### Chrome
- Go to [Settings](chrome://settings)
- On the left sidebar, hit **On startup**
- Select **Open a specific page or set of pages**
- Click **Add a new page**
- In the textbox, enter **https://shankar-sachin.github.io/ai-speed-dial**
- Refresh Chrome to see your beautiful new landing page!
#### Others (Firefox, Brave, Opera, etc.)
- If Chromium-based, follow **Chrome** instructions
- If not, navigate to your browser's settings
- Search **On startup** in the search box
- Then type in **https://shankar-sachin.github.io/ai-speed-dial** to the textbox
- Refresh the browser to see your beautiful new landing page!
